# 🏠 House Price Prediction Using Machine Learning

## 📌 Project Overview
This project aims to predict house prices using machine learning models based on demographic and geographic features.

## 🚀 Models Used
- Linear Regression
- Random Forest

## 📊 Results
| Model              | R² Score | MAE  |
|--------------------|--------|------|
| Linear Regression  | 0.57   | 0.53 |
| Random Forest      | 0.80   | 0.32 |

✅ Random Forest performed significantly better.

## 💡 Key Insight
Median income is the most important factor influencing house prices.
